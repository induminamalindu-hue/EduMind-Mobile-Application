# 🎓 EduMind - Student Attendance & Academic Management System

![Android](https://img.shields.io/badge/Platform-Android-green.svg)
![Kotlin/Java](https://img.shields.io/badge/Language-Java%2FKotlin-orange.svg)
![HND Project](https://img.shields.io/badge/Project-HND%20Final%20Project-blue.svg)

**EduMind** is a comprehensive mobile application developed as the final project for the **Higher National Diploma (HND) in Information Technology**. The system is designed to streamline academic administrative tasks, enabling seamless track of student attendance, marks management, learning materials distribution, and secure authentication.

---

## ✨ Features

### 👤 Authentication & Security
* **Multi-role Login:** Secure user login for Students, Faculty, and Administrators.
* **OTP Verification:** Enhanced security with OTP verification during account registration/recovery.
* **reCAPTCHA Integration:** Protection against automated bot interactions.

### 📊 Attendance Management
* **QR / Code Scanning Attendance:** Quick attendance marking using mobile scan functionality.
* **Faculty Attendance Tracker:** Lecturers can mark and review daily class attendance.
* **Attendance Reports:** Real-time generation of student attendance statistics and reports.

### 📚 Academic & Marks Management
* **Marks Portal:** Lecturers can input, update, and manage student evaluation marks.
* **Learning Materials:** Access and download subject course materials easily.
* **Dashboard:** Interactive dashboard presenting real-time stats for both students and faculty.

---

## 🛠️ Tech Stack & Architecture

* **Language:** Java / Kotlin
* **IDE:** Android Studio
* **UI Components:** ViewBinding, Material Design Components, RecyclerView
* **Architecture:** MVC / MVVM Pattern
* **Build System:** Gradle (v8.13)

## ⚙️ Installation & Setup

1. **Clone the Repository:**
   ```bash
   git clone [https://github.com/YOUR_GITHUB_USERNAME/EduMind-Android-App.git](https://github.com/YOUR_GITHUB_USERNAME/EduMind-Android-App.git)
   ````
## Open in Android Studio:

⚫ Open Android Studio.

⚫ Select Open an Existing Project.

⚫ Browse to the cloned EduMind-Android-App directory.

## Build & Sync:

⚫ Allow Gradle to sync dependencies automatically.

⚫ Ensure you have Android SDK (API Level 21 or higher) installed.

## Run the Application:

⚫ Connect an Android device with Developer Mode enabled OR start an Emulator.

⚫ Click Run (Shift + F10).
